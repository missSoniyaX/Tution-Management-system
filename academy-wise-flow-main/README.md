1. Project Overview
Tution Management system is a web-based Tuition and Academy Management System developed for small coaching classes and tuition centers. The system helps manage students, teachers, attendance, fees, notes, notices, and syllabus progress digitally instead of using manual paperwork.
The project provides separate modules for Admin, Teacher, and Student users with secure login access.
________________________________________
2. Objectives of the Project
•	To digitize coaching class management
•	To reduce manual paperwork
•	To automate student fee management
•	To provide online access to notes and materials
•	To manage attendance digitally
•	To track syllabus completion
•	To improve communication between academy and students

3. How to Run the Project
Step 1 – Install Node.js
Install Node.js from: https://nodejs.org

Step 2 – Extract Project
Extract the ZIP file.

Step 3 – Open Terminal
Open terminal inside the project folder.

Step 4 – Install Dependencies
Run:
npm install

Step 5 – Start Development Server
Run:
npm run dev

Step 6 – Open Browser
Open:
http://localhost:5173
